# Gold Layer Data Catalog

## Overview

The Gold layer contains curated business-facing views designed for reporting, KPI tracking, dashboarding, and analytical use cases.

The views in this layer are built on top of the Silver layer and combine cleaned operational, passenger, baggage, gate, and retail data into reusable structures.

The Gold layer currently contains:

* Dimension views for descriptive business entities
* Fact views for measurable business activity
* Derived metrics, rankings, aggregations, and business classifications

---

# 1. gold.dim_flights

## Purpose

Stores descriptive flight-level attributes used for operational reporting and airline performance analysis.

## Grain

One row per flight.

## Columns

| Column Name                         | Data Type | Description                                                         |
| ----------------------------------- | --------- | ------------------------------------------------------------------- |
| flight_key                          | BIGINT    | Surrogate key assigned to each flight record in the dimension view. |
| flight_id                           | VARCHAR   | Business identifier for the flight.                                 |
| airline_name                        | VARCHAR   | Name of the airline operating the flight.                           |
| airline_code                        | VARCHAR   | Airline code associated with the carrier.                           |
| origin_airport                      | VARCHAR   | Departure airport or city name.                                     |
| destination_airport                 | VARCHAR   | Arrival airport or city name.                                       |
| route_name                          | VARCHAR   | Combined route name in origin-destination format.                   |
| route_type                          | VARCHAR   | Route classification such as Domestic or International.             |
| aircraft_type                       | VARCHAR   | Aircraft model used for the flight.                                 |
| aircraft_registration               | VARCHAR   | Aircraft registration number.                                       |
| terminal                            | VARCHAR   | Terminal assigned to the flight.                                    |
| gate_number                         | VARCHAR   | Gate assigned to the flight.                                        |
| season                              | VARCHAR   | Seasonal category associated with the flight.                       |
| departure_day_of_week               | VARCHAR   | Day of week for departure.                                          |
| arrival_time_of_day                 | VARCHAR   | Time-of-day category for arrival.                                   |
| international_flight_flag           | TINYINT   | Indicates whether the flight is international.                      |
| weekend_flight_flag                 | TINYINT   | Indicates whether the flight departed on a weekend.                 |
| flight_status                       | VARCHAR   | Current operational status of the flight.                           |
| delay_category                      | VARCHAR   | Delay severity category based on departure delay minutes.           |
| departure_delay_minutes             | INT       | Number of minutes the flight departed late.                         |
| flight_duration_minutes             | INT       | Total flight duration in minutes.                                   |
| seat_capacity                       | INT       | Maximum number of seats available on the aircraft.                  |
| booked_passengers                   | INT       | Number of passengers booked on the flight.                          |
| recalculated_load_factor_percentage | DECIMAL   | Derived passenger load factor percentage for the flight.            |
| ticket_revenue                      | DECIMAL   | Total ticket revenue generated by the flight.                       |

# 2. gold.dim_passengers

## Purpose

Stores passenger-level descriptive attributes for customer segmentation, demographic analysis, and travel behavior reporting.

## Grain

One row per passenger.

## Columns

| Column Name        | Data Type | Description                                                          |
| ------------------ | --------- | -------------------------------------------------------------------- |
| passenger_key      | BIGINT    | Surrogate key assigned to each passenger record.                     |
| passenger_id       | VARCHAR   | Unique passenger identifier.                                         |
| loyalty_number     | VARCHAR   | Loyalty membership number associated with the passenger.             |
| passenger_name     | VARCHAR   | Full passenger name created from first and last name.                |
| nationality        | VARCHAR   | Nationality of the passenger.                                        |
| gender             | VARCHAR   | Standardized gender value.                                           |
| date_of_birth      | DATE      | Passenger date of birth.                                             |
| passenger_age      | INT       | Passenger age in years.                                              |
| age_group          | VARCHAR   | Derived age group classification.                                    |
| travel_class       | VARCHAR   | Cabin class selected by the passenger.                               |
| booking_channel    | VARCHAR   | Method used to book the flight.                                      |
| vip_flag           | TINYINT   | Indicates whether the passenger is marked as VIP.                    |
| no_show_flag       | TINYINT   | Indicates whether the passenger did not show up for the flight.      |
| satisfaction_score | DECIMAL   | Passenger satisfaction rating.                                       |
| passenger_segment  | VARCHAR   | Derived segment classification based on VIP status and travel class. |

| age_group                          | VARCHAR   | Passenger age group category.                                      |
| nationality                        | VARCHAR   | Passenger nationality.                                             |
| booking_channel                    | VARCHAR   | Booking source used by the passenger.                              |
| vip_flag                           | TINYINT   | Indicates whether the passenger is VIP.                            |
| no_show_flag                       | TINYINT   | Indicates whether the passenger was a no-show.                     |
| baggage_count                      | INT       | Number of baggage items associated with the passenger.             |
